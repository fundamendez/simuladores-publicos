# 🚀 Centro de Simulación Interactiva

Una colección de herramientas visuales diseñadas para la materia **Fundamentos de la Programación** (Cátedra Méndez).

Este hub reúne distintos entornos interactivos para visualizar en tiempo real y facilitar la comprensión de conceptos complejos sobre memoria, algoritmos y estructuras de datos.

### Simuladores incluidos:

- **Simulador de Memoria RAM:** Visualización de alojamiento de variables, punteros y arreglos contiguos. **[Link](https://fundamendez.github.io/simuladores-publicos/simulador-memoria-ram/index.html)**
- **Emulador de Ordenamientos:** (En desarrollo). **[Link](https://fundamendez.github.io/simuladores-publicos/simulador-ordenamientos/index.html)**
- **Simulador de Búsquedas:** (En desarrollo). **[Link](https://fundamendez.github.io/simuladores-publicos/simulador-busquedas/index.html)**
- **Simulador de TDAs:** (En desarrollo). **[Link](https://fundamendez.github.io/simuladores-publicos/simulador-tdas/index.html)**
