# 🧠 Simulador Interactivo de Memoria RAM

Una herramienta visual diseñada para la materia **Fundamentos de la Programación** (Cátedra Méndez).

Este simulador permite visualizar en tiempo real cómo se alojan las variables en la memoria, ejemplificando conceptos complejos como punteros, direcciones hexadecimales y arreglos contiguos.

🔗 **[Link al simulador](https://fundamendez.github.io/simulador-memoria-ram/)**
